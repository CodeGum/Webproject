Metadata Tags Description Example (Put "< >" Around)
base Specifies a default URL/target for all of the links on a page, must be no more than one base href="https://www.example.com/"
link Defines the relationship between the current document and an external resource, can only be used in the head section link rel="link-type" href="URL"
meta Provides structured metadata (keywords, descriptions, author info, etc.), can only be used in the head section meta name="keywords" content="HTML,CSS,Tutorials,References,Examples"
style Tells the browser how to present the document, can only be used in the head section style type="text/css" ... /style
title Defines the title of the document to be displayed in the web browser title ... /title
List Tags Description Example (Put "< >" Around)
dd Represents the description/definition of an item in a description list dd ... /dd
dl Specifies a definition list dl ... /dl
dt Represents a term/item in a definition list dt ... /dt
li Defines an individual list item within a list li ... /li
ol Defines an ordered list of items ol ... /ol
menu Defines a list of commands a user can perform menu ... /menu
menuitem In use with menu, defines each item in the menu list menuitem /
ul Defines an unordered list of items ul ... /ul
Table Tags Description Example (Put "< >" Around
caption Defines the title of a table, only one can be used per table caption ... /caption
col Represents one or more columns, can apply style rules to an entire column col
colgroup Defines a group of one or more columns within a table colgroup ... /colgroup
table Represents data within rows and columns table ... /table
tbody Groups a set of rows defining the main body of the table data tbody ... /tbody
td Defines a data cell in a table td ... /td
tfoot Defines a table's footer, typically used to group a set of table rows summarizing the columns of the table tfoot ... /tfoot
thead Defines a table's header, typically used to group a set of table rows defining the header of the columns within the table thead ... /thead
th Defines a header cell in a table th ... /th
tr Defines a row of cells in a table tr ... /tr
