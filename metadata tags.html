Metadata tags

Metadata Tags Description Example (Put "< >" Around)
base Specifies a default URL/target for all of the links on a page, must be no more than one base href="https://www.example.com/"
link Defines the relationship between the current document and an external resource, can only be used in the head section link rel="link-type" href="URL"
meta Provides structured metadata (keywords, descriptions, author info, etc.), can only be used in the head section meta name="keywords" content="HTML,CSS,Tutorials,References,Examples"
style Tells the browser how to present the document, can only be used in the head section style type="text/css" ... /style
title Defines the title of the document to be displayed in the web browser title ... /title
