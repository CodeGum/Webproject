Formatting tags

Tag Description Example
<abbr> Defines an abbreviated form of a longer word or phrase. Used to to show the longer version of an abbreviated word when hovering over it.
such as<abbr title="Hyper Text Markup Language">HTML</abbr>
<address> Specifies the author's contact information. This tag does the same effect as <i> or <em> but it conveys additional semantic information
<b> Displays text in a bold style. Makes text like this <b>bold</b>
<bdi> Represents text that is isolated from its surrounding for the purposes of bidirectional text formatting. Used to display that text that is normally right to left is going left to right, such as arabic
<bdo> Overrides the current text direction. The sequence "1-2-3-4-5" was reversed as: 1-2-3-4-5
<blockquote> Represents a section that is quoted from another source.
<cite> Indicates a citation or reference to another source. Used to tag text as a <cite>citation</cite>, normally rendered in italics but can be changed with CSS
<code> Specifies text as computer code.

<dfn> Specifies a definition. Used to show that a word is being defined such as:
